Time Selector

This plugin turns input fields into time selectors. The concept is borrowed from the iCal time stamp formatter. Each field has many keyboard shortcuts to enable a better experience. The left and right arrow switch the currently focused subfield. The up and down arrows increment the time up or down. The mousewheel also allows the user to navigate through time. The keypad may also be used to manually enter the time.

The formatter allows for a flexible set of controls. By default the format 'm/d/Y h:ia' is used.
